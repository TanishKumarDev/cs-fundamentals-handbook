# **Octal Number System (Base 8)**

## 1. What is Octal?

* Base: 8
* Digits: 0, 1, 2, 3, 4, 5, 6, 7
* Origin: "Oct" = Eight (Latin)
* Each octal digit corresponds to 3 binary digits, making it a compact form of binary.

---

## 2. Place Value in Octal

* Rightmost digit = 8^0
* Next = 8^1
* Next = 8^2 … and so on

Example:
(247)_8 = 2 × 8^2 + 4 × 8^1 + 7 × 8^0 = 128 + 32 + 7 = 167_10

---

## 3. Conversions

### Octal → Decimal

1. Multiply each digit by its place value (powers of 8) from right to left.
2. Sum all results.

Example: (123)_8 → Decimal
1 × 8^2 + 2 × 8^1 + 3 × 8^0 = 64 + 16 + 3 = 83

---

### Octal → Binary

1. Convert each octal digit into 3-bit binary.
2. Combine all bits.

Example: (247)_8 → Binary
2 → 010, 4 → 100, 7 → 111 → Combine → 010100111_2

---

### Octal → Hexadecimal

1. Convert octal → binary (each octal digit → 3 bits)
2. Group binary into 4 bits from right, add leading zeros if needed
3. Convert each group → hexadecimal digit

---

### Decimal → Octal

1. Divide decimal number by 8
2. Write remainder
3. Repeat until quotient = 0
4. Write remainders in reverse → octal number

Example: Decimal 100 → Octal
100 ÷ 8 = 12 r 4
12 ÷ 8 = 1 r 4
1 ÷ 8 = 0 r 1 → Octal = 144_8

## 5. Solved Examples

1. Octal → Decimal
   (1121)_8 = 1 × 8^3 + 1 × 8^2 + 2 × 8^1 + 1 × 8^0 = 512 + 64 + 16 + 1 = 593_10

2. Octal → Binary
   (27)_8 → 2 → 010, 7 → 111 → 010111_2

3. Binary → Octal
   (10101111)_2 → group 3 bits → 001 010 111 → 1 2 7 → 127_8

4. Octal → Decimal
   (123)_8 = 1 × 8^2 + 2 × 8^1 + 3 × 8^0 = 64 + 16 + 3 = 83_10

---

## 6. Quick Conversion Tricks

* Octal → Binary: each digit → 3-bit binary
* Binary → Octal: group 3 bits from right → convert
* Octal → Hex: Octal → Binary → Hex
