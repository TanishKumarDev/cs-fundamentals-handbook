# Constructors and Member Functions

## Default Constructors in C++

* **Definition:**
  A default constructor is a special member function automatically called when an object of a class is created and **takes no arguments**.

* **Types:**

  1. **Implicitly Generated by Compiler:** If no constructor is defined, the compiler provides a default constructor.
  2. **Explicitly Defined by Programmer:** Programmer defines a constructor with no parameters.
  3. **With Initialization:** Can initialize member variables inside the constructor body.

---

### **Example 1: No User-Defined Constructor**

```cpp
class GFG {
public:
    // Compiler provides default constructor automatically
};
GFG obj;  // Default constructor called implicitly
```

---

### **Example 2: Explicitly Defined Default Constructor**

```cpp
class GFG {
public:
    GFG() { }  // Programmer-defined default constructor
};
GFG obj;  // Calls the explicitly defined constructor
```

---

### **Example 3: Default Constructor with Initialization**

```cpp
#include <iostream>
using namespace std;

class GFG {
    int x;
public:
    GFG() {
        x = 10;
        cout << "x initialized to " << x << endl;
    }
};

int main() {
    GFG obj;  // Default constructor called
    return 0;
}
```

**Output:**

```
x initialized to 10
```

---

### **Use Cases of Default Constructor**

* Creating **arrays or vectors** of objects.
* Required when using **STL containers** (`vector<Class>`).
* Objects need **default initialization values**.
* Important in **generic programming** and **templates**.

---

### **Example: Default Constructor with Members**

```cpp
#include <iostream>
using namespace std;

class GFG {
    string name;
    int roll;
public:
    GFG() {  // Default constructor
        name = "Unknown";
        roll = -1;
        cout << "Default constructor called\n";
    }

    void display() {
        cout << "Name: " << name << ", Roll: " << roll << endl;
    }
};

int main() {
    GFG s1;  // Calls default constructor
    s1.display();
    return 0;
}
```

**Output:**

```
Default constructor called
Name: Unknown, Roll: -1
```
